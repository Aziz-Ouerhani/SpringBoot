Application de gestion de consultations et de documents

Ce projet est une application Spring Boot qui permet de gérer les consultations, les documents associés, et les utilisateurs.
Elle offre une structure complète pour organiser et suivre les interactions entre les utilisateurs (patients, médecins ou administrateurs), tout en assurant la gestion sécurisée des informations et des fichiers liés aux consultations.

 Fonctionnalités principales

Gestion des utilisateurs : création, authentification et attribution de rôles.

Gestion des consultations : ajout, mise à jour, suppression et consultation des rendez-vous.

Gestion des documents : association de documents médicaux ou administratifs à une consultation spécifique.

Architecture RESTful : toutes les opérations sont accessibles via une API REST claire et bien structurée.

Sécurité : authentification et autorisation basées sur Spring Security (JWT ou session selon configuration).
